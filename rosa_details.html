About project

🌺 Rosa Shop offers a smooth and powerful shopping experience with smart tools for both customers and admins 🎯.

Technology Used

  • PHP
  • LARAVEL
  • lIVEWIRE
  • MYSQL

Features

    🛠 Admin Dashboard Features:

  • ✅ Manage products, categories, and orders
  • ✅ Send emails using SMTP with queue jobs for better speed 📧⚙
  • ✅ Set shipping costs, apply discounts, manage users & roles
  • ✅ Easily update permissions with role-based access control 🔐
  • ✅ Receive email notifications when a new order is placed by a user 📥📢
  • 📡 Hosted on Hostinger with a custom subdomain 🌍
  • 🛠 Built with passion, precision, and a lot of learning! ❤‍🔥

    • 🧑‍🤝‍🧑 User Features:

  • 🛒 Add to cart and favorites using Livewire (no page reload 🔄)
  • 📬 Receive email notifications for offers and updates
  • 📨 Contact us through a form (sends real emails via SMTP)
  • 💳 Place orders using Cash on Delivery or Online Payment (via Paymob)
  • 🔐 Secure login with Google Integration
  • 🔎 View top/best-selling products
  • 🔍 Search for products and filter by category/section for easy browsing
